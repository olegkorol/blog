This is just a reference post to see how we can use the chessground and pgn-viewer components from Lichess in my blog.

Here are a few boards using [Chessground](https://github.com/lichess-org/chessground) with different options (all view-only for the blog).

### 1. Opening position (default)

Plain starting position. No extra config.

<Chessboard config={{}} />

### 2. Custom position with check

A specific FEN, turn color, and the king in check highlighted (`check: true`).

<Chessboard config={{
  fen: 'r1bqkbnr/1ppppBpp/p1n5/8/4P3/8/PPPP1PPP/RNBQK1NR b KQkq - 0 1',
  orientation: 'white',
  turnColor: 'black',
  check: true,
}} />

### 3. Last move highlighted

After 1.e4, the moved squares are highlighted via `lastMove` and `highlight.lastMove`.

<Chessboard config={{
  fen: 'rnbqkbnr/pppppppp/8/8/4P3/8/PPPP1PPP/RNBQKBNR b KQkq e3 0 1',
  lastMove: ['e2', 'e4'],
  highlight: { lastMove: true },
}} />

### 4. Black’s perspective

Same position as above but with `orientation: 'black'` so the board is from Black’s side.

<Chessboard config={{
  fen: 'rnbqkbnr/pppppppp/8/8/4P3/8/PPPP1PPP/RNBQKBNR b KQkq e3 0 1',
  orientation: 'black',
  lastMove: ['e2', 'e4'],
  highlight: { lastMove: true },
}} />

### 5. Coordinates on squares

File and rank labels on each square (`coordinatesOnSquares: true`) and ranks on the left (`ranksPosition: 'left'`).

<Chessboard config={{
  coordinatesOnSquares: true,
  ranksPosition: 'left',
}} />

### 6. Arrows and labels (drawable)

Pre-drawn shapes: arrows and text labels via `drawable.autoShapes` (green/red/blue brushes, optional labels).

<Chessboard config={{
  fen: 'r2q2k1/1p6/p2p4/2pN1rp1/N1Pb2Q1/8/PP1B4/R6K b - - 2 25',
  drawable: {
    visible: true,
    autoShapes: [
      { orig: 'g4', dest: 'g7', brush: 'red', label: { text: '!' } },
      { orig: 'd5', dest: 'f6', brush: 'green' },
      { orig: 'd2', dest: 'h6', brush: 'blue' },
    ],
  },
}} />

### 7. Selected square

A square can be marked as selected via `selected` (e.g. to show the user’s pick). Works with `highlight` for the selected-square styling.

<Chessboard config={{
  fen: 'rnbqkbnr/pppppppp/8/8/4P3/8/PPPP1PPP/RNBQKBNR b KQkq e3 0 1',
  selected: 'e4',
  lastMove: ['e2', 'e4'],
  highlight: { lastMove: true },
}} />

### 8. No coordinates

Edge coordinates can be turned off with `coordinates: false` for a clean, border-only board.

<Chessboard config={{
  coordinates: false,
}} />

### 9. Circles (drawable without destination)

When a shape has only `orig` and no `dest`, Chessground draws a circle on that square. Handy for highlighting key squares.

<Chessboard config={{
  fen: 'r1bqkb1r/pppp1ppp/2n2n2/4p2Q/2B1P3/8/PPPP1PPP/RNB1K1NR w KQkq - 4 4',
  drawable: {
    visible: true,
    autoShapes: [
      { orig: 'e5', brush: 'yellow' },
      { orig: 'f6', brush: 'red' },
      { orig: 'h5', brush: 'green' },
    ],
  },
}} />

### 10. Check and last move together

You can combine `check`, `lastMove`, and `highlight.lastMove` / `highlight.check` in one position.

<Chessboard config={{
  fen: 'r1bqkbnr/pppp1ppp/2n5/4p2Q/2B1P3/8/PPPP1PPP/RNB1K1NR b KQkq - 0 3',
  turnColor: 'black',
  check: true,
  lastMove: ['d1', 'h5'],
  highlight: { lastMove: true, check: true },
}} />

### 11. Labels only (no brush)

Shapes can be label-only: use `label` without `brush` to show text in the top-right of the square (useful for move numbers or comments).

<Chessboard config={{
  fen: 'rnbqkbnr/pppppppp/8/8/4P3/8/PPPP1PPP/RNBQKBNR b KQkq e3 0 1',
  drawable: {
    visible: true,
    autoShapes: [
      { orig: 'e4', label: { text: '1' } },
      { orig: 'e5', label: { text: '1…' } },
    ],
  },
}} />

### 12. Ranks on the right + coordinates on squares

With `coordinatesOnSquares: true`, rank labels can sit on the right via `ranksPosition: 'right'` (the default when not set).

<Chessboard config={{
  coordinatesOnSquares: true,
  ranksPosition: 'right',
  fen: '4k3/8/8/8/8/8/8/4K3 w - - 0 1',
}} />

## PGN Viewer

The [Lichess PGN Viewer](https://github.com/lichess-org/pgn-viewer) lets you step through full games with move notation, player info, and navigation controls.

### 13. Immortal Game (Anderssen vs Kieseritzky, 1851)

One of the most famous games ever played. Step through the brilliancies with the controls below the board.

<PgnViewer pgn={`[Event "London"]
[Site "London"]
[Date "1851.06.21"]
[White "Adolf Anderssen"]
[Black "Lionel Kieseritzky"]
[Result "1-0"]

1. e4 e5 2. f4 exf4 3. Bc4 Qh4+ 4. Kf1 b5 5. Bxb5 Nf6 6. Nf3 Qh6
7. d3 Nh5 8. Nh4 Qg5 9. Nf5 c6 10. g4 Nf6 11. Rg1 cxb5 12. h4 Qg6
13. h5 Qg5 14. Qf3 Ng8 15. Bxf4 Qf6 16. Nc3 Bc5 17. Nd5 Qxb2
18. Bd6 Bxg1 19. e5 Qxa1+ 20. Ke2 Na6 21. Nxg7+ Kd8 22. Qf6+ Nxf6
23. Be7# 1-0`}
  showMoves="right"
/>

### 18. Viewer without controls

A minimal embed — moves panel only, no player info, no arrow drawing.

<PgnViewer pgn={`[Event "Miniature"]
[White "NN"]
[Black "NN"]
[Result "*"]

1. d4 d5 2. c4 e6 3. Nc3 Nf6 4. Bg5 Be7 5. e3 O-O 6. Nf3 Nbd7 *`}
  showControls={false}
  showPlayers={false}
  drawArrows={false}
  showMoves="bottom"
/>
